Клиент

При нажатии на кнопку "Войти" вызывается функция ConnectClicked (из UIManager) - происходит подключение к серверу.
Когда подключение произошло, вызывается функция DidConnect (NetworkManager) - отправляются на сервер логин и пароль.
Ответ о проверке логина и пароля обрабатывается функцией RecieveLogPas (NetworkManager) - открывается следующее окно меню.
Из меню[1] происходит вход в игру: принажатии на кнопку вызывается функция SendEnterGameRequest - отправляется сообщение с номером персонажа, которым хочет играть пользователь.
